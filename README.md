# Java Quiz Application

A simple console-based quiz application written in Java.  
The app shows multiple choice questions, takes user input from the terminal, and calculates the final score based on correct answers.

## Features

- Predefined set of multiple-choice questions  
- Options displayed for each question  
- User selects an option using the keyboard  
- Checks the answer and maintains score  
- Displays the result at the end of the quiz

## Files in this project

- `Main.java` – Entry point of the application; starts the quiz.  
- `Question.java` – Model class that stores a single question, its options, and the correct answer.  
- `QuestionService.java` – Contains the list of questions and the logic to display them and evaluate answers.

## How to run

1. Make sure Java (JDK) is installed on your system.  
2. Clone this repository: https://github.com/apurv99-Cloud/Quiz-Application.git
